# Virtual Fragrance Shelf

Welcome to the **Virtual Fragrance Shelf**, a premium, editorial-style inventory management system and social platform for perfume enthusiasts.

**Live Application:** [https://personal-perfume.vercel.app/](https://personal-perfume.vercel.app/)

## Features

- **Interactive 3D-styled Shelf UI**: A stunning visual display of your fragrance collection.
- **AI-Powered Organization**: Utilize Google's Gemini API to dynamically organize and enrich your collection's metadata.
- **Master Wall View**: A dense, seamless grid display of all perfumes for a curated experience.
- **Admin Panel**: Robust, secure, and production-ready administration tools for managing platform-wide metadata consistency.
- **Community Features**: Connect with other enthusiasts and explore trending collections.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Database**: Supabase
- **Styling**: Tailwind CSS
- **AI Integration**: Google Gemini API
- **Image Storage**: Cloudinary

## Getting Started
